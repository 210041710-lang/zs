# Boss直聘客户端自动化招聘 使用文档

## 1. 项目定位

`Boss直聘客户端自动化招聘` 是一个围绕 Boss 直聘客户端使用场景搭建的自动化招聘系统，提供以下能力：

- AI 分析岗位并生成 JD、关键词和筛选标准
- 自动执行候选人搜索、打招呼、跟进和简历收集
- 对简历进行 AI 评分并筛选达标候选人
- 对达标候选人自动跟进联系方式
- 在前端看板中实时查看任务状态、漏斗和日志

## 2. 启动方式

### 后端

在项目根目录下进入后端目录：

```bash
cd /Users/Admin/Documents/boss-hiring-v2-main1/backend
python3 main.py serve --port 8001
```

如果 `8000` 端口已被占用，建议继续使用 `8001`。

### 前端访问

浏览器打开：

- [http://127.0.0.1:8001](http://127.0.0.1:8001)

## 3. 页面说明

### 作战总览

首页用于统一查看：

- 候选人总数
- 已收简历数
- 达标人数
- 已获取联系方式人数
- 当前运行任务的招聘漏斗
- 实时执行日志
- 全部任务列表

适合用于日常巡检和判断任务是否需要暂停、继续或停止。

### 新建任务

在“新建任务”页面中：

1. 输入职位名称和补充说明
2. 点击 AI 分析
3. 检查和编辑关键词
4. 配置每日上限、评分阈值、工作时段等参数
5. 启动任务

### 候选人池

在“候选人池”页面中可以：

- 按状态筛选候选人
- 查看初筛分数和简历评分
- 查看是否已获取联系方式
- 导出 Excel

### 职位矩阵

在“职位矩阵”页面中可以：

- 查看已创建的岗位画像
- 查看每个岗位累计关联的候选人数
- 快速判断哪些岗位值得继续投放自动化招聘

### 市场情报

该页面用于查看市场调研数据，包括公司、岗位和人才画像等内容。

### 系统设置

该页面用于：

- 配置模型接入信息与运行参数
- 测试 LLM 连通性
- 添加或重新登录 Boss 直聘账号

## 4. 推荐使用流程

建议按以下顺序使用：

1. 进入“系统设置”配置 LLM Key 和模型
2. 在“系统设置”中登录 Boss 直聘账号
3. 进入“新建任务”创建招聘任务
4. 回到“作战总览”观察漏斗和实时日志
5. 在“候选人池”复盘结果并导出数据

## 5. 注意事项

- 首次运行前请确认后端 `.env` 已配置正确
- 若模型接口不可用，先到“系统设置”测试连接
- 若 Boss 直聘账号未登录，自动化任务可能无法正常执行
- 若页面无法访问，优先检查后端服务是否仍在运行
- 对外分享仓库前，请确认不要提交任何真实密钥、服务地址或账号数据
